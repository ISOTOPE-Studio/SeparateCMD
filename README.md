# SperateCMD
将命令分段输入

## Download/下载:
https://github.com/ISOTOPE-Studio/SperateCMD/releases

## 使用方法
输入命令是将需要单独输入的地方用 !arg 代替

例子：

`/manuaddp !arg essentials.warp`

`/tp !arg`

`/op !arg`

例子：

玩家被迫或主动输入：`/tp !arg

插件提醒玩家玩家输入需要传送的玩家

玩家输入Mars

之后就会以玩家的身份输入 `/tp Mars`

## 配置文件

玩家必须在以下时间内输入命令，一秒为单位

`timeout: 3.0`
