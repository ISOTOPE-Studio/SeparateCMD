# SperateCMD
将命令分段输入

## Download/下载:
https://github.com/ISOTOPE-Studio/SperateCMD/releases

## 使用方法
输入命令是将需要单独输入的地方用 !arg 代替，并且在最后加上由谁执行命令（player或console）

例子：

`/manuaddp !arg essentials.warp console`

`/tp !arg player`

`/op !arg console`

### 玩家

例子：

玩家被迫或主动输入：`/tp !arg player`

插件提醒玩家玩家输入需要传送的玩家

玩家输入Mars

之后就会以玩家的身份输入 `/tp Mars`
